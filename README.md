Selenium Practice Project

Overview

This repository contains my Selenium WebDriver automation practice project developed using Python. The project automates various web elements and functionalities available on the Test Automation Practice website.

Website Automated

https://testautomationpractice.blogspot.com/

Technologies Used

- Python
- Selenium WebDriver
- PyCharm
- Git & GitHub

Test Scenarios Covered

- Text box input handling
- Radio button selection
- Checkbox selection
- Dropdown handling
- Date picker interaction
- Web table handling
- Alert handling
- Browser navigation
- Element identification using XPath, CSS Selectors, ID, and Name locators

Project Structure

SeleniumPracticeProject1/

└── README.md

How to Run

1. Clone the repository:

git clone https://github.com/your-username/SeleniumPracticeProject1.git

2. Install Selenium:

pip install selenium

3. Run the Python script:

python main.py
